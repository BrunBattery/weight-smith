# ⚖️ Weight Smith

A minimal web app for use by lifters. Built with modern web technologies and designed for mobile-first use.

---

## ✨ Features

### 🔢 Plate Calculator
- **Visual plate loading** - Color-coded plates showing exactly what to load
- **Barbell or machine mode** - Toggle between 45lb barbell or machine pin loading
- **Smart rounding** - Automatically rounds to nearest 2.5lb increment
- **Interactive controls** - Slider, +/- buttons, or direct input
- **Real-time breakdown** - See exactly which plates to use per side

---

## 🚀 Technology

React + TypeScript • Vite • Tailwind CSS • PWA

---

## 📦 Installation

### As a Web App
Visit the deployed site and use it directly in your browser.

### Install as PWA
1. Visit the app in Safari (iOS) or Chrome (Android)
2. Tap "Share" → "Add to Home Screen"
3. Use like a native app
